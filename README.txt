1) node.js installieren (https://nodejs.org/en/download/)
2) commandline öffnen
3) packages "socket.io" und "express" installieren
   -cd C:/projectdirectory
   -npm install socket.io express --save
4) server starten
   -node serversocket.js
5) browser öffnen: localhost:8080
6) es sollte index.html angezeigt werden.

PS: sollte es die Verzeichnisse "temp" und "Video" im projektverzeichnis noch nicht geben, bitte anlegen!
